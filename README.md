Usage
-----

1. Clone it
2. Give it a name
3. Create something awesome

Development
-----------

### Run specs with [RSpec]

Run `rspec`.

or via [Guard]:

```
$ guard -g spec
```

### Run [RuboCop]

Run `rubocop`.

To run all specs and RuboCop altogether, run `rake`.
